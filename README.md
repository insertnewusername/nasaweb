<h1>Nasaweb</h1>

Nasaweb is a website that fetches the image of the day from nasa and then displays it on the website.

Simply open the site, and it'll fetch nasa's image of the day and display it to you! There will also be a paragraph explaning the image below for you to understand.
This project was highly dependent on the stardance tutorial, helping me learn about apis, and vite.

This project is for stardance!

Setting up is fairly simple. Just open the website, and it'll do all the work for you!
